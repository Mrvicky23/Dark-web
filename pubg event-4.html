BATTLEGROUND MOBILE INDIA

Battlegrounds Mobile India (abbreviated BGMI) is an online multiplayer mobile battle royale for Indian players, developed and published by KRAFTON





The player will be disqualified if found using any kind of emulator. Any game modifying tools except 'GFX tool' is not allowed. Players can play on android/ios tablets/phones only. Only in-game voice chat should be used after the game is started till its completion.
